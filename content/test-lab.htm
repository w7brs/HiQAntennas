Hi-Q-Antennas Test Lab

"One TEST is worth a thousand OPINIONS"

We measure our HF Mobile antenna performance with high-end test equipment, to back up what we advertise!

Test Equipment used:

  • HP-4191A LCR-Q meter
  • HP-4342A Q meter
  • HP-8558B Spectrum Analyzer
  • HP 8591A Spectrum Analyzer
  • HP-8405A Vector Volt meter
  • HP-5434A Frequency Counter
  • HP-8640A Frequency Generator
  • HP 8753C Network Analyzer with
    85046A "S" Parameter Unit.
  • Boonton 260A Q meter

The eight charts presented were made on the AEA VIA antenna analyzer.
Measurements where taken on the Hi-Q-4/80 HF mobile antenna on the test vehicle.
Test frequency was 3810 kc.

This combined chart shows four parameters:
    (S) VSWR
    (Z) Impedance
    (X) Reactance
    (R) Resistance
